全民K歌房间状态查询系统

Tencent KTV Room Status Query System

使用帮助：

  • 1、歌房ID如何获取？打开手机全民K歌APP，进入要查询的歌房，选择分享按钮并点击复制链接，然后查找链接中的roomid字段等号后的内容，复制并输入到本页面即可！如：39d9db84232b328d314a71
  • 2、程序支持电脑端在线观看么？暂时没有准备做这个功能，但是已经发现了腾讯的直播服务器接口。如果你感兴趣，可以联系我。
  • 3、本程序的接口开放调用么？目前开放，接口地址：https://api.dingstudio.cn/dingstudio/qktv_room.php?roomId=xxxxx 后期不排除由于大量请求而开启相关身份验证功能，我会在这里公告。
  • 4、程序开源问题：本程序将在稍候在本人的GitHub、Coding和私有Git服务器开放源代码。支持开源奉献精神，从你我做起！欢迎广大开发者进行二次封装。
